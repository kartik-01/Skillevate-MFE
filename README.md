# Skillevate MFE

This repository contains the front end code for skillevate


# Requirements:

Following should be installed in your system for your project to run smoothly.
* Python
* Node.js
* React 

# Steps to Run:

Once the repository is cloned in your system & you have the folder in you system unzipped.

1. Open Terminal
2. Navigate to the folder
3. Run the command ./start.sh

All the related services would start running.
